# FinalT2Assignmentcpp
This is the final assignment for VFS T2 assignment as C++ code. It is a Pet simulator
By Cris and Yeison

# === Goal === #
Train your pet up and be the best pet racer of all time.
Balance your stats with survival and mood.
Try your best to not make your pet die / run away from you!

# === Goal of Code === #
The goal of this project was to gain experience in C++ and experience in working as a team.
We tried our best following a single coding convention and tried to suggestion improvements to each other.
Hope you enjoy.

# === Install Instructions === #
Just download as a zip and run the sln file.
after that just run Source.cpp and play the game.
